# 量化选股策略 - A股超跌买入策略

## 项目简介

本项目是一个基于Python的A股量化选股策略工具，主要实现超跌买入策略。通过结合KDJ指标和MACD指标，筛选出处于超卖状态且有反弹迹象的股票。

## 选股策略

本策略基于以下三个核心条件进行选股：

1. **KDJ指标J值为负数** - 表示股票处于超卖区域
2. **MACD金叉或即将金叉** - 表示趋势可能反转向上
3. **前2个月跌幅超过20%** - 表示股票处于超跌状态

只有同时满足以上三个条件的股票才会被选中。

## 使用方法

### 选股程序（实时选股）

`ash
python main.py
`

### 回测程序（验证历史表现）

`ash
python backtest_main.py
`

### 运行模式

程序提供两种运行模式：

1. **测试模式** - 只分析前100只股票，适合快速测试和验证
2. **全量模式** - 分析所有A股股票，耗时较长但覆盖面全

首次运行建议使用测试模式，确认策略效果后再使用全量模式。

### 选股程序输出

程序运行后会生成以下文件（保存在output目录）：

- 选股结果.csv - 符合条件的股票列表
- 全部分析结果.csv - 所有分析过的股票数据
- 选股报告.txt - 格式化的选股报告
- strategy.log - 运行日志

### 回测程序功能

回测程序用于验证策略的历史表现，可以选择：

1. **回测日期**：选择过去某个日期，模拟在那天选股
2. **持仓周期**：1周 / 1个月 / 2个月
3. **回测指标**：胜率、平均收益、最大盈利、最大回撤

回测程序输出：

- 回测结果.csv - 各股票的详细回测数据
- 回测报告.txt - 汇总报告（包含胜率等关键指标）
- backtest.log - 回测日志

## 策略参数配置

可以在config/config.py中调整策略参数：

`python
STRATEGY_CONFIG = {
    # KDJ参数
     kdj_period: 9,        # KDJ周期
    kdj_signal: 3,        # KDJ平滑周期
    
    # MACD参数
    macd_fast: 12,        # MACD快线周期
    macd_slow: 26,        # MACD慢线周期
    macd_signal: 9,       # MACD信号线周期
    
    # 超跌条件
    decline_period: 42,   # 跌幅计算周期（交易日，约2个月）
    decline_threshold: -0.20,  # 跌幅阈值（-20%）
}
`

## 技术指标说明

### KDJ指标

KDJ指标是一种技术分析工具，由K值、D值和J值组成：
- K值：快速确认线
- D值：慢速确认线
- J值：方向敏感线

J值为负数表示股票处于极度超卖状态，是潜在的买入信号。

### MACD指标

MACD（移动平均收敛发散指标）由以下组成：
- DIF：差离值（快线）
- DEA：讯号线（慢线）
- MACD柱：DIF与DEA的差值

MACD金叉（DIF上穿DEA）是买入信号，表示趋势可能由跌转涨。

## 回测说明

回测是验证量化策略有效性的重要方法。本项目的回测逻辑：

1. 选择一个历史日期（如30天前）
2. 模拟在那天运行选股策略
3. 计算选出的股票在之后N天的涨跌情况
4. 统计胜率（盈利股票占比）和平均收益

**回测周期说明**：
- 1周（5个交易日）- 短期表现
- 1个月（21个交易日）- 中期表现
- 2个月（42个交易日）- 长期表现

## 常见问题

### 程序运行很慢怎么办？

获取股票数据需要一定时间，特别是全量模式。建议：
- 先使用测试模式验证
- 确保网络连接正常
- 耐心等待数据获取完成

### 没有选出任何股票怎么办？

这是正常现象，说明当前市场没有同时满足所有条件的股票。可以：
- 检查策略参数是否过于严格
- 降低跌幅阈值（如改为-15%）
- 等待市场出现超跌机会

### 如何修改选股条件？

可以在config/config.py中修改参数，或者直接修改strategy/oversold_buy.py中的策略逻辑。
