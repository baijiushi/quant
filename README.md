# 量化选股策略 - A股超跌买入策略

## 项目简介

本项目是一个基于Python的A股量化选股策略工具，主要实现超跌买入策略。通过结合KDJ指标和MACD指标，筛选出处于超卖状态且有反弹迹象的股票。

## 选股策略

本策略基于以下三个核心条件进行选股：

1. **KDJ指标J值为负数** - 表示股票处于超卖区域
2. **MACD金叉或即将金叉** - 表示趋势可能反转向上
3. **前2个月跌幅超过20%** - 表示股票处于超跌状态

只有同时满足以上三个条件的股票才会被选中。

## 环境要求

- Python 3.8+
- 依赖包（见requirements.txt）

## 安装步骤

1. 克隆或下载本项目到本地

2. 安装依赖包：
`
pip install -r requirements.txt
`

## 使用方法

### 运行程序

`
python main.py
`

### 运行模式

程序提供两种运行模式：

1. **测试模式** - 只分析前100只股票，适合快速测试和验证
2. **全量模式** - 分析所有A股股票，耗时较长但覆盖面全

首次运行建议使用测试模式，确认策略效果后再使用全量模式。

### 输出结果

程序运行后会生成以下文件（保存在output目录）：

- 选股结果.csv - 符合条件的股票列表
- 全部分析结果.csv - 所有分析过的股票数据
- 选股报告.txt - 格式化的选股报告
- strategy.log - 运行日志
